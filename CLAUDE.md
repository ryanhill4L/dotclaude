### Interaction Style
Always ask clarifying questions before starting implementation on ambiguous or complex requests. Do NOT assume scope or approach - confirm with the user first via the AskUserQuestion tool.

### Plan vs Execute
When asked to 'execute a plan', only execute what is described in the plan document. Do not expand scope beyond the plan.

When asked to write a plan or PRD, deliver only the document - do not begin implementation unless explicitly asked.

Use an agent team to execute on plans unless explicitly told NOT to. The main agent should delegate all tasks to teammates.
